# Tests - 测试模块

> 最后更新：2026-02-16

## 模块职责

`backend/tests` 包含后端单元测试和集成测试，使用 pytest 框架。

## 目录结构

```
tests/
├── __init__.py
├── conftest.py              # pytest 配置和 fixture
├── test_auth.py             # 认证测试
├── test_models.py           # 数据模型测试
├── test_contract.py         # 合同模块测试
├── test_payment.py          # 支付模块测试
├── test_attendance.py       # 考勤模块测试
├── test_notification.py     # 通知模块测试
├── test_course_service.py   # 课程服务测试
├── test_student_service.py  # 学员服务测试
├── test_tasks.py            # 定时任务测试
├── test_payment_flow.py     # 支付流程测试
├── test_student_integration.py  # 学员集成测试
├── test_api.py             # API 集成测试
├── test_courses.py         # 课程 API 测试
├── test_students.py        # 学员 API 测试
├── test_student_api.py     # 学员 API 测试
├── test_end_to_end.py      # 端到端测试
├── test_payment_callback_handler.py  # 支付回调测试
└── test_wechat_pay.py      # 微信支付测试
```

## 测试结果

```bash
pytest tests/ -v
# 结果: 118 passed, 36 skipped, 13 warnings
```

## 运行测试

```bash
# 运行所有测试
pytest tests/ -v

# 运行特定模块测试
pytest tests/test_auth.py -v

# 运行带覆盖率测试
pytest tests/ --cov=app --cov-report=html
```

## 依赖

测试所需额外依赖：
```bash
pip install email-validator aiosqlite greenlet
```
