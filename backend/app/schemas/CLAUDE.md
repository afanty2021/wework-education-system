# Schemas - Pydantic 数据模式

> 最后更新：2026-02-16

## 模块职责

`backend/app/schemas` 包含所有 Pydantic 数据验证模式，用于 API 请求/响应数据验证。

## 目录结构

```
schemas/
├── __init__.py
├── user.py                 # 用户模式
├── student.py             # 学员模式
├── course.py              # 课程模式
├── contract.py            # 合同模式
├── payment.py             # 支付模式
├── schedule.py            # 排课模式
├── attendance.py          # 考勤模式
├── homework.py           # 作业模式
└── notification.py        # 通知模式
```

## Schema 模式清单

| Schema 文件 | 主要类 | 说明 |
|-------------|--------|------|
| user.py | UserCreate, UserUpdate, UserResponse | 用户 CRUD 模式 |
| student.py | StudentCreate, StudentUpdate, StudentResponse | 学员 CRUD 模式 |
| course.py | CourseCreate, CourseUpdate, CourseResponse | 课程 CRUD 模式 |
| contract.py | ContractCreate, ContractUpdate, ContractResponse | 合同 CRUD 模式 |
| payment.py | PaymentCreate, PaymentUpdate, PaymentResponse | 支付 CRUD 模式 |
| schedule.py | ScheduleCreate, ScheduleUpdate, ScheduleResponse | 排课 CRUD 模式 |
| attendance.py | AttendanceCreate, AttendanceUpdate, AttendanceResponse | 考勤 CRUD 模式 |
| homework.py | HomeworkCreate, HomeworkUpdate, HomeworkResponse | 作业 CRUD 模式 |
| notification.py | NotificationCreate, NotificationUpdate, NotificationResponse | 通知 CRUD 模式 |

## Pydantic v2 配置

所有 Schema 使用 `model_config = ConfigDict(...)` 配置：

```python
from pydantic import ConfigDict

class StudentCreate(ModelBase):
    name: str
    phone: str
    model_config = ConfigDict(str_strip_whitespace=True)
```
