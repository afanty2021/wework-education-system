# CRUD - 数据访问层

> 最后更新：2026-02-16

## 模块职责

`backend/app/crud` 包含数据库 CRUD 操作，基于 SQLModel 提供数据访问接口。

## 目录结构

```
crud/
├── __init__.py
├── base.py                 # 基础 CRUD 类
├── user.py                 # 用户 CRUD
├── student.py              # 学员 CRUD
├── course.py               # 课程 CRUD
├── contract.py             # 合同 CRUD
├── payment.py              # 支付 CRUD
├── schedule.py             # 排课 CRUD
├── attendance.py           # 考勤 CRUD
└── notification.py         # 通知 CRUD
```

## CRUD 类清单

| CRUD 文件 | CRUD 类 | 对应模型 |
|-----------|---------|----------|
| base.py | CRUDBase | 基础模板 |
| user.py | UserCRUD | User |
| student.py | StudentCRUD | Student |
| course.py | CourseCRUD, ClassroomCRUD, DepartmentCRUD | Course, Classroom, Department |
| contract.py | ContractCRUD | Contract |
| payment.py | PaymentCRUD, RefundCRUD | Payment, Refund |
| schedule.py | ScheduleCRUD | Schedule |
| attendance.py | AttendanceCRUD | Attendance |
| notification.py | NotificationCRUD | Notification |

## 使用示例

```python
from app.crud.student import StudentCRUD
from app.database import get_session

async def get_students():
    async for session in get_session():
        crud = StudentCRUD(session)
        return await crud.get_all()
```
