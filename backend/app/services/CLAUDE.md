# Services - 业务服务层

> 最后更新：2026-02-16

## 模块职责

`backend/app/services` 包含所有业务逻辑服务，提供数据访问层和 API 路由之间的业务处理。

## 目录结构

```
services/
├── __init__.py
├── auth_service.py          # 认证服务
├── wework_service.py       # 企业微信服务
├── student_service.py       # 学员服务
├── course_service.py       # 课程服务
├── contract_service.py     # 合同服务
├── payment_service.py      # 支付服务
├── attendance_service.py   # 考勤服务
├── schedule_service.py     # 排课服务
└── notification_service.py # 通知服务
```

## 服务清单

| 服务文件 | 服务类 | 职责 |
|----------|--------|------|
| auth_service.py | AuthBusinessService | 用户认证、JWT Token 处理 |
| wework_service.py | WeWorkService | 企业微信 API 集成 |
| student_service.py | StudentService | 学员 CRUD 操作 |
| course_service.py | CourseService | 课程管理业务逻辑 |
| contract_service.py | ContractService | 合同管理、课时扣减 |
| payment_service.py | PaymentService | 支付流程处理 |
| attendance_service.py | AttendanceService | 考勤记录管理 |
| schedule_service.py | ScheduleService | 排课管理 |
| notification_service.py | NotificationService | 通知推送 |
