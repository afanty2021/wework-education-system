# API - 前端 API 请求模块

> 最后更新：2026-02-16

## 模块职责

`frontend/admin/src/api` 包含所有后端 API 请求封装。

## API 清单

| 文件 | API 模块 | 说明 |
|------|----------|------|
| request.ts | axios 实例 | HTTP 请求封装 |
| auth.ts | 认证 API | 登录、Token 刷新 |
| students.ts | 学员 API | 学员 CRUD |
| courses.ts | 课程 API | 课程 CRUD |
| contracts.ts | 合同 API | 合同 CRUD |
| payments.ts | 支付 API | 支付 CRUD |
| schedules.ts | 排课 API | 排课 CRUD |
| attendance.ts | 考勤 API | 考勤 CRUD |
| notifications.ts | 通知 API | 通知 CRUD |

## 使用示例

```typescript
import { studentApi } from '@/api'

// 获取学员列表
const students = await studentApi.getList()

// 创建学员
await studentApi.create({ name: '张三', phone: '13800138000' })
```
